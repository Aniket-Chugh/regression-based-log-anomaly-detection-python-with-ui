# Regression Based Log Anomaly Detection Python With User Interface 

A Streamlit app to detect anomalies in log data using regression.

## 1. Clone the Repository

```bash
git clone https://github.com/Aniket-Chugh/regression-based-log-anomaly-detection-python-with-ui.git
```

## 2. Navigate to Project Folder

```bash
cd regression-based-log-anomaly-detection-python-with-ui
```

## 3. Create Virtual Environment

```bash
python -m venv .venv
```

## 4. Activate Virtual Environment

### Windows (PowerShell)

```powershell
.venv\Scripts\Activate.ps1
```

## 5. Install Dependencies

```bash
pip install -r requirements.txt
```

## 6. Run the Application 🚀

```bash
python -m streamlit run app.py
```

## 7. Open in Browser

```
http://localhost:8501
```
